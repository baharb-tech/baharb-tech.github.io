Introduction

  • Summary statistics are the first step in understanding any dataset.

  • They help us see:

    • Typical values (mean, median)

    • How data is spread (variance, SD)

    • Patterns and anomalies

    • Possible errors or outliers

Why Summary Statistics Matter

  • Summary statistics help us quickly understand large datasets.
  • They show:
    • Central values (mean, median)
    • Spread of data (range, variance, standard deviation)
    • Distribution shape (skewness, kurtosis)
  • They help detect:
    • Outliers
    • Incorrect values
    • Missing data
    • Data entry mistakes
  • They make analysis faster, simpler, and more accurate.

Types of Summary Statistics

1. Measures of Central Tendency

  • Mean ‚Äì average value
  • Median ‚Äì middle value
  • Mode ‚Äì most frequent value

2. Measures of Spread

  • Range ‚Äì difference between max and min
  • Variance ‚Äì how far values are from the mean
  • Standard Deviation ‚Äì average spread of data

3. Measures of Shape

  • Skewness ‚Äì direction of tilt

  • Kurtosis ‚Äì how peaked or flat the distribution is

4. Count & Frequency

  • Count of observations
  • Frequency tables

Techniques to Compute Summary Statistics in R

Base R: Simple Summary Statistics

Example Dataset

Heights of 5 students:

heights <- c(160, 165, 170, 155, 180)

Base R Summary Statistics

mean(heights)      # average height
median(heights)    # middle value
sd(heights)        # standard deviation
summary(heights)   # full summary

dplyr Summary Statistics

library(dplyr)

df <- data.frame(heights)

df %>%
  summarise(
    mean_height = mean(heights, na.rm = TRUE),
    median_height = median(heights, na.rm = TRUE),
    sd_height = sd(heights, na.rm = TRUE)
  )

Explanation

  • summarise() creates new summary values.
  • Each line inside summarise calculates one statistic.
  • na.rm = TRUE removes missing values before calculation.

Grouped Summary Statistics

library(dplyr)

# Example dataset
df_grouped <- data.frame(
  group = c("A", "A", "B", "B", "B"),
  value = c(10, 12, 8, 15, 14)
)

# Grouped summary
df_grouped %>%
  group_by(group) %>%
  summarise(
    avg_value = mean(value),
    count     = n()
  )

Explanation

  • group_by(group) splits the dataset into categories (A and B).
  • summarise() calculates summary statistics inside each group.
  • This helps compare groups easily (e.g., group A vs group B).
